## Features:
- Restream
- Transcode (CPU/GPU)
- Logo overlay
- Text overlay
- Computer vision
- WebRTC streams
- Change sources in realtime (AD insertion)
- Stream statistics
- Video effects
- Virtual background
- Many Outputs
- Physical Inputs (/dev/video, IP Camera, DVB-T/C/S, ATSC, ISDB-T and DTMB)
- Channel on demand
- Record video
